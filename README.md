# Fortune Push

*Get your fortune paper by git push!*


### install
```terminal
go install github.com/zett-8/fp
```

### how to use

just replace 'git push' command with 'fp'
```terminal
fp origin master
```

That's it!  
Every time you push to Github using 'fp' command you'll get your fortune paper!!
